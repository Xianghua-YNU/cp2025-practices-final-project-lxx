# 行星状星云分析系统

## 功能概述
本系统用于分析行星状星云数据，包括：
- FITS文件加载与预处理
- 深度特征提取
- 聚类分析
- 光学与射电数据匹配
- 结果可视化

## 文件结构
- `main.py`: 主程序入口
- `data_processing/`: 数据处理模块
- `analysis/`: 分析模块
- `matching/`: 数据匹配模块
- `visualization/`: 可视化模块

## 环境配置
```bash
pip install -r requirements.txt
```

## 运行方式
```bash
python main.py
```
